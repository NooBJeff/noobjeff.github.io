CentOS的一些(ge)使用经验

公司的服务器运行CentOS 7，因为有些命令我也不知道是不是Linux通用的，所以标题不敢写成 Linux， 只好放个 CentOS。

  1. 强制结束进程：ctrl + \ 一般用 ctrl + c 就能终止大部分程序的运行，但是有些程序，比如CentOS的yum，就会忽略这个信号。

    按下 ctrl + c 后会有一个 “Keyboard Interrupt” signal 发送到当前程序。但是程序如何响应该信号，是终止自己还是忽略，操作系统管不了。
    同样的，按下ctrl + \ 或者ctrl+4 后会有一个 “Quit” signal，这个通常意味着当前程序需要尽快退出，甚至跳过保存。大概相当于非常紧急的信息，应该没有程序会选择忽略。
